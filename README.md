**8 free, single-file tools** for SaaS founders. Deploy anywhere in 2 minutes.

## Features

- **Zero dependencies** — Single HTML files, no build step
- **Zero backend** — Runs entirely in browser, no data stored
- **Mobile responsive** — Works on all devices
- **Instant deployment** — GitHub Pages, Netlify, Vercel
- **Branded** — "Free tool by SimpleRecover" footer on each

## Deployment

### GitHub Pages (Free)
1. Create a repo: `yourname.github.io/seo-tools`
2. Upload these HTML files
3. Enable GitHub Pages in settings
4. Done — `https://yourname.github.io/seo-tools/`

### Netlify Drop (Free)
1. Go to app.netlify.com/drop
2. Drag and drop this folder
3. Get instant CDN URL

### Vercel (Free)
```bash
npm i -g vercel
vercel --prod
```
